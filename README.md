# exercicio_ebac-3.2
exercicios da ebac do curso qualidade de software


Este repositório contém um projeto focado em Garantia de Qualidade (QA), desenvolvido para consolidar práticas essenciais de testes automatizados e manuais. O objetivo principal é garantir a confiabilidade de uma aplicação através de uma pirâmide de testes bem estruturada.

Destaques do Projeto:
Testes E2E: Automação de fluxos críticos utilizando Cypress/Selenium para validar a experiência do usuário.

Testes de API: Validação de endpoints e contratos com foco em performance e segurança.

Cobertura de Código: Implementação de testes unitários para garantir a integridade da lógica de negócio.

Relatórios: Geração de logs detalhados e evidências de bugs para facilitar o processo de correção.

Boas Práticas: Uso de Clean Code e Page Objects Pattern (POM) para manter o código escalável.
